# Machine Learning Basics

Este repositório contém exemplos práticos de **aprendizado de máquina** utilizando Python puro, abordando conceitos fundamentais de **Regressão, Classificação e Clustering**.

O objetivo é servir como referência para estudo e prática, mostrando de forma simples como funcionam os modelos, funções de perda e atualização de parâmetros com **Gradient Descent**.

---

## Conteúdo

### 1. Regressão
- Modelo linear com múltiplas variáveis.
- Cálculo de previsões (`y_hat`) com pesos (`w`) e bias (`b`).
- Funções de perda:
  - **MSE** (Mean Squared Error)
  - **MAE** (Mean Absolute Error)
  - **Huber Loss**
- Exemplo de atualização de pesos via **Gradient Descent**.

### 2. Classificação
- Avaliação de modelos com:
  - **0-1 Loss** (taxa de erro)
  - **Log Loss** (função logarítmica para probabilidades)
- Exemplos com valores reais e predições simuladas.

### 3. Gradient Descent
- Demonstração do cálculo de gradiente para atualização de pesos.
- Exemplo de convergência de um modelo de regressão linear simples.
